Project ต้นแบบ 3: ระบบจัดการธนาคารขยะอัจฉริยะ (Waste Bank) ด้วย Line OA AI

โครงการนี้พัฒนาโซลูชันสำหรับธนาคารขยะ โดยใช้ Line Official Account (Line OA) เป็นช่องทางหลัก ในการสื่อสารกับสมาชิก และใช้ AI Automation เพื่อจัดการข้อมูลและกระบวนการ.

ฟังก์ชันการทำงานหลัก (Features)

Line OA User Interface

สมาชิกธนาคารขยะสามารถแจ้งข้อมูลขยะ, สอบถามยอดสะสม, และรับข่าวสารผ่าน Line OA.

Automated Data Processing

ข้อมูลที่สมาชิกแจ้งผ่าน Line OA ถูกบันทึกและคำนวณยอดสะสมอัตโนมัติด้วย n8n/AppScript.

Waste Data Analytics

แสดงสถิติประเภทและปริมาณขยะที่รวบรวมได้ใน AppSheet/Google Sheet Dashboard.

*
Personalized Notifications

แจ้งเตือนสมาชิกอัตโนมัติ (ยอดสะสม, โปรโมชั่น, กิจกรรม) ผ่าน Line Notify หรือ Email.
